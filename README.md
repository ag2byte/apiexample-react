# Simple api handler

This is a simplke example of handling APIs in React using axios

### npm packages:

- axios
- bootstrap
- reactstrap
- reacticons

### How to run

- create a new react app using

```
npx create-react-app apihandle
```

- replace the folders given there with that from the repo
- run it using

```
npm start
```
